# Analise-Exploratoria-de-dados-com-Python

# 📊 Análise Exploratória de Dados - Titanic

Este projeto realiza uma Análise Exploratória de Dados (EDA) utilizando o dataset clássico do Titanic, com o objetivo de identificar padrões, relações e fatores que influenciaram a sobrevivência dos passageiros.

---

## 🔗 Fonte dos dados

* Dataset: Titanic (Kaggle / dataset público)

---

## 🛠️ Tecnologias utilizadas

* **Python** → análise e manipulação de dados
* **Pandas** → tratamento e exploração dos dados
* **Matplotlib / Seaborn** → visualizações
* **Jupyter Notebook** → desenvolvimento da análise

---

## 🚀 Como utilizar

1. Clone este repositório ou faça o download dos arquivos
2. Instale as dependências:

   ```bash
   pip install -r requirements.txt
   ```
3. Abra o notebook:

   ```bash
   jupyter notebook
   ```
4. Execute as células para reproduzir a análise

---

## 📊 Etapas da análise

* Carregamento e entendimento dos dados
* Tratamento de valores ausentes
* Análise univariada
* Análise bivariada
* Criação de visualizações
* Extração de insights

---

## 📌 Informações analisadas

* Distribuição de idade dos passageiros
* Sobrevivência por sexo
* Sobrevivência por classe (Pclass)
* Relação entre idade e sobrevivência
* Impacto da tarifa (Fare)
* Influência de variáveis categóricas

---

## 💡 Principais insights

* Mulheres apresentaram maior taxa de sobrevivência
* Passageiros da 1ª classe tiveram maior probabilidade de sobreviver
* Crianças tiveram maior taxa de sobrevivência em comparação com adultos
* Tarifas mais altas estão associadas a maior taxa de sobrevivência
* A classe social foi um dos fatores mais determinantes

---

## 📈 Exemplos de análises

### Sobrevivência por sexo

Mostra uma diferença significativa na taxa de sobrevivência entre homens e mulheres.

### Classe social e sobrevivência

Evidencia como passageiros da 1ª classe tiveram vantagem.

### Idade e sobrevivência

Permite identificar padrões de sobrevivência entre diferentes faixas etárias.
